📝 Todo App

A simple and interactive Todo App built using HTML, CSS, and JavaScript. This application helps users manage their daily tasks efficiently by adding, viewing, and deleting tasks with ease.

🚀 Features
➕ Add new tasks with a date
📅 Date selection for each task
🗑️ Delete tasks easily
📋 Display task list dynamically
🎯 Clean and user-friendly UI
🛠️ Tech Stack
HTML – Structure of the app
CSS – Styling and layout
JavaScript – Functionality and DOM manipulation
